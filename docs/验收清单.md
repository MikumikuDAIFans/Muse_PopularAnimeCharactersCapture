# Muse DataLoad 性能优化与自动榜单更新验收清单

Last updated: 2026-05-15 00:31 Asia/Shanghai

## 验收总则

- 本清单对应 `docs/开发计划.md` 中的 `muse-dataload-performance-weekly-ranking` 任务书。
- 所有性能结论必须包含数据规模、执行命令、运行环境、耗时和输出证据。
- 自动更新默认只处理 Danbooru 元数据和榜单，不下载图片。
- 历史 PostgreSQL-only 验收结果作为基础能力已通过；本清单关注新的性能优化和每周增量更新能力。
